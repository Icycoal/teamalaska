# InCollege - (COBOL)



## How to Compile and Run

1. Make sure you have a COBOL compiler installed (e.g., **GnuCOBOL**).  

2. Compile the program:

```bash
cobc -x -free InCollege.cob -o InCollege
```

2. Run program:

```bash
./InCollege   
```
